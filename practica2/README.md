## Configuracion de Router 1, 2, 5

### Router 1

```
enable
conf t
no ip domain-lookup
hostname R1
```
```
int se0/0
ip add 10.0.0.1 255.255.255.252
no shut
exit
```
```
int f0/0

no shut
```
